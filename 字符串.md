# 最小表示法

优化主要思想：若两字符串前k-1位都一样，第k位不一样，那么下一次指针可以直接移动k+1位，而不用再一位一位移动

```c++
int k = 0, i = 0, j = 1;
while (k < n && i < n && j < n) {
  if (sec[(i + k) % n] == sec[(j + k) % n]) {
    k++;
  } else {
    //主要优化点
    sec[(i + k) % n] > sec[(j + k) % n] ? i = i + k + 1 : j = j + k + 1;	//若求最大表示把这句改成大于即可
    if (i == j) i++;
    k = 0;
  }
}
i = min(i, j);
```
